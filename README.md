# PMP Academy 4.1 — Clean Development Foundation

This package is a clean replacement for the `development` branch.

## Important

Delete or replace the old development-branch files. The repository root should contain only:

- index.html
- styles.css
- app.js
- manifest.webmanifest
- sw.js
- README.md
- .nojekyll
- icon-192.png
- icon-512.png

There are no JSON files or nested data folders in this clean deployment. All curriculum, lesson, question, and activity data is embedded in `index.html`, which avoids iPhone folder-upload problems.

## Included

- Data-driven curriculum architecture
- Chapter 1 migrated into the lesson engine
- Chapter 2 foundation started
- Checkpoints throughout the lessons
- Drag-and-drop matching with tap fallback
- Sequencing activity
- Practice questions
- Progress tracking
- Independent Version 4.1 browser storage
- Offline PWA support
